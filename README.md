# Smart-Home
自己实现的智能家居，包括语音交互，远程控制，web访问等功能

##### 使用
    1. python 3.5
    2. php 5.6
    3. mysql 5.6
    4. laravel php框架 5.3

### 依赖库
    1. pycurl
    2. pygame
    3. numpy
    4. pyaudio
    5. wave
    6. MySQL-python驱动 (pip3 install mysql-connector-python-rf --allow-external mysql-connector-python)
